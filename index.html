BookShelf is a web application built using Symfony framework for managing books, authors, and publishers. Key features include:

User Authentication: Users can register and login with email verification.

Role-based Access Control: Users can browse books, while authenticated users can add, edit, and delete their own books. Admins have full control, including user role management.

Book Management: Users can add, edit, and delete books. When adding or editing, authors and publishers are associated accordingly.

Author and Publisher Management: Seamless management of authors and publishers, ensuring accurate association with books.

Pagination: Efficient pagination for navigating through the book collection.

Database Integration: The system integrates with a relational database (e.g., MySQL, PostgreSQL) for data storage and retrieval, facilitating robust backend functionality.

Technology Stack: Symfony Framework, PHP, Twig Templating Engine, Doctrine ORM, KnpPaginatorBundle, Symfony Forms.

The system provides a user-friendly, fully responsive on all screens interface for exploring and managing books, making it suitable for both casual readers and enthusiasts.
